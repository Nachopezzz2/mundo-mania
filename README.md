# Mundo Manía — Retail Store Website

A bold, vibrant website for a multi-category retail store in Carmelo, Uruguay.

## 🔗 Live Demo
[mundo-mania.vercel.app](https://mundo-mania.vercel.app)

## 🛠️ Built With
- Next.js 14
- TypeScript
- Tailwind CSS

## ✨ Features
- Product catalog with 8+ categories (toys, tools, stationery, gifts, etc.)
- New arrivals section with product cards
- Category filtering
- WhatsApp contact integration
- Store hours and location
- Social proof (5.6K Facebook / 3.2K Instagram followers)

## 📁 Project Context
Freelance project for a family retail business with 8+ product categories
in Carmelo, Uruguay.
